## Overview

v1.0 交付了「二次元个人网站（工具库 + 追番记录）」三大核心模块端到端可用的最小产品：
- Astro 7 + Tailwind v4 静态站脚手架，改文件 → git push → Vercel CDN 上线全链路
- 鲜艳可爱二次元外壳与首页（亮/暗双主题，防 FOUC，响应式）
- 工具库核心：卡片列表 / MDX 详情 / 分类档案 / 标签档案 / 安全外链
- 追番记录：卡片列表 / 在看·看完·想看 状态筛选 / 详情 hero（评分/进度/短评）

## Milestone Summary

**Key Decisions:**

- 采用 Astro + Tailwind v4 静态站，改文件 + git 自动部署（无后台/无数据库）
- Tailwind v4 经 `@tailwindcss/vite`（CSS-first @theme），跳过已弃用的 `@astrojs/tailwind`
- 内容用 Content Layer API（content.config.ts + glob loader + Zod）
- 番剧 schema 分层（客观元数据组含 bgmId 预留 + 追番状态组永远手动），数据读取收口 lib/anime.ts，为 Bangumi API 迁移铺路
- 主题初始应用走 `<head>` 内联同步脚本防 FOUC；岛屿最小化，默认全静态 .astro
- EntryCard 双层链接走 article.relative + 绝对定位外链（零嵌套 a，D-04 / TOOL-05）
- 站点标题走 ZCOOL KuaiLe 手工子集 woff2（4.3KB）而非完整字体
- 工具库详情排版走 @layer utilities .prose hand-rolled（~40 行）而非第三方 typography 插件

**Issues Resolved:**

- code-review HIGH-#1 false-positive revert（CJK decoded params+href），加 review status frontmatter 兜底
- 01-02 落地页 push 块受阻临时代理抖动，已本地 commit 待网络恢复

**Issues Deferred:**

- 推送全部本地提交（含 01-02 落地页 + 01-03 SEO 占位页 + phase-2 规划物）到 origin main — 受网络代理抖动影响
- 生产站部署尚未反映最新本地提交（本地领先 origin）

**Technical Debt Incurred:**

- EntryCard 整卡单 `<a>` 在 externalUrl 未传时保留 Phase 1 行为；Phase 2 升级 article.relative 双层链接由 02-02 处理 ✓，旧调用点依赖 externalUrl 守卫
- AnimeCard 封面无图时用 Blob SVG 占位，后续接真实封面路径或 Bangumi 远程图需补 image 配置
- .prose hand-rolled 仅覆盖 h2/h3/p/ul/a/code/pre/img/blockquote/hr，长文需求再补（tables/figures 等）
