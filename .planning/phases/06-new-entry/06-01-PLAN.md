**客户端 `<script define:vars={{ existingSlugs }}>`**：

核心逻辑：
1. slug 自动派生：titleCn 输入 → `input` 事件 → `encodeURIComponent(val)` → 填入 slug 输入框
2. slug 冲突检测：slug 输入框 `input` 事件 → 检查 `existingSlugs.includes(val)` → 冲突则加 error class
3. 校验：同 Phase 5 模式
4. 生成 frontmatter：组装 fields 对象 → 序列化为 YAML 字符串（`---` 包裹）+ JSON 对象
5. 预览：显示在 `<pre>` 中
6. 复制：`navigator.clipboard.writeText(frontmatterStr)`
7. 下载 JSON：`const blob = new Blob([jsonStr], {type: 'application/json'}); const url = URL.createObjectURL(blob); const a = document.createElement('a'); a.href = url; a.download = slug + '.json'; a.click(); URL.revokeObjectURL(url);`

Frontmatter 生成函数（伪代码）：
```javascript
function generateFrontmatter() {
  const fields = {
    titleCn: titleCnEl.value.trim(),
    titleJa: titleJaEl.value.trim() || undefined,
    status: statusEl.value,
    progress: Number(progressEl.value) || 0,
    episodes: episodesEl.value ? Number(episodesEl.value) : undefined,
    myRating: ratingEl.value ? Number(ratingEl.value) : undefined,
    comment: commentEl.value.trim() || undefined,
    cover: coverEl.value.trim() || undefined,
    airDate: airDateEl.value || undefined,
    draft: false,
  };
  // YAML
  let yaml = '---\n';
  Object.entries(fields).forEach(([k, v]) => {
    if (v === undefined) return;
    if (typeof v === 'string') yaml += `${k}: "${v}"\n`;
    else yaml += `${k}: ${v}\n`;
  });
  yaml += '---\n\n';
  yaml += bodyEl.value;
  // JSON
  const json = JSON.stringify({ ...fields, body: bodyEl.value }, null, 2);
  return { yaml, json };
}
```
