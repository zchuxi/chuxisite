# Phase 6 — Plan 02: 工作台新增按钮

## Goal

在 `/admin` 工作台追番/工具网格上方各加一行「新增」按钮，点击跳转至新增条目页。

## Verification

1. `npm run build` exit 0
2. `/admin` 页面「追番」网格上方有「+ 新增追番」按钮
3. 点击按钮跳转至 `/admin/new/anime`
4. 「工具」网格上方有「+ 新增工具」按钮
5. 点击按钮跳转至 `/admin/new/tools`

**type:** auto

**action:**

修改 `src/pages/admin/index.astro`，在每个 `<section data-grid>` 开头各插入一行新增按钮。

**追番网格**（`<section data-grid="anime">` 开头，在 EmptyState 判断之前）：
```astro
<div class="mb-4">
  <a href="/admin/new/anime" class="inline-flex min-h-11 items-center gap-1 rounded-[var(--radius-md)] bg-[var(--color-primary)] px-4 text-sm font-semibold text-[var(--color-primary-fg)] transition hover:bg-[var(--color-primary-hover)]">

  </a>
</div>
```

**工具网格**（`<section data-grid="tools">` 开头，在 EmptyState 判断之前）：
```astro
<div class="mb-4">
  <a href="/admin/new/tools" class="inline-flex min-h-11 items-center gap-1 rounded-[var(--radius-md)] bg-[var(--color-primary)] px-4 text-sm font-semibold text-[var(--color-primary-fg)] transition hover:bg-[var(--color-primary-hover)]">
    + 新增工具
  </a>
</div>
```
