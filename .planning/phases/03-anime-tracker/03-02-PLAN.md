# 03-02 — AnimeCard / StatusPill / RatingStar + 追番列表主页（wave 2）

## Goal

交付「追番列表主页」端到端可用：封面网格（1→2→3 列）+ 顶部状态 pill 行（全部/在看/做完/想看）+ summary line + 空状态。新建 `AnimeCard`（封面 + 标题 + 状态标签 + 评分 + 进度，整卡可点）、`StatusPill`（卡片内小标签，封面右上角绝对定位）、`RatingStar`（卡片上精简数字 `9/10`）。覆盖 ANIME-01（列表卡 5 项）、ANIME-02（状态 pill 入口跳静态筛选页）、ANIME-04（消费 status/progress/myRating/episodes/comment 字段）。

## must_haves（由 phase goal 派生）

- `src/pages/anime/index.astro` 从 Phase 1 占位页**迁出**为真实列表页：Header / Footer / BaseLayout 壳；summary line `共 {N} 部 · 慢补番中～`；PillRow 状态 pill 行（全部→`/anime`、在看→`/anime/status/watching`、做完→`/anime/status/done`、想看→`/anime/status/plan`，active `aria-current="page"`）；1→2→3 列封面网格（`grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-6 lg:gap-8`）；空则 EmptyState（Copywriting 锁定文案）。
- `AnimeCard.astro` 整卡 `<a href={href}>` 包裹封面 + 信息区；封面 `aspect-[2/3]`、`rounded-[var(--radius-lg)]`、`object-cover`、`loading="lazy"`；cover 缺失走 Blob 占位（青粉渐变 + 浮动星心，aria-hidden，CLS≈0）；信息区含 titleCn（单行 ellipsis）、StatusPill（兄弟绝对定位，不进入 `<a>` 嵌套）、RatingStar（精简数字）、进度文字；整卡 hover `motion-safe:-translate-y-0.5` + 边框变 `--color-primary`；focus-visible 2px primary ring + 2px offset。
- `StatusPill.astro`：`rounded-[var(--radius-sm)] px-2 py-0.5 text-xs font-semibold`；配色 watching→primary 染底+primary 字、done→success 染底+success 字、plan→deco-pink 染底+deco-pink 字（color-mix ~10-12%）；卡片内 `absolute right-2 top-2 z-10 shadow-sm`，**作为整卡 `<a>` 的兄弟遮罩**（仿 EntryCard 外链按钮方案，避免嵌套冒泡）。
- `RatingStar.astro`：卡片上**精简数字** `9/10`（UI-SPEC Planner Notes 决策：卡片空间有限，5 star 冗余）；`aria-hidden`（整卡 `<a>` 的 aria-label 已含评分）；数字取 `--color-text-muted`、star 图标取 `.star-fill`。
- 数据访问**全部经** `src/lib/anime.ts` 的 `getAllAnime()` / `slugOf()`，不直接 `getCollection`（ANIME-05）。
- 零新 npm 依赖、零 Preact 岛屿。

## Decision Coverage

- **D-04**：新建 AnimeCard（不复用 EntryCard，信息结构差异大）；双层链接模式参考 EntryCard（`article.relative` + 整卡 `<a>` + 兄弟绝对定位 StatusPill）。
- **D-05**：列表网格 1→2→3 列；容器 `max-w-[72rem]` 居中。
- **D-06**：卡上展示全部 5 项（封面 + 标题 + 状态标签 + 评分 + 进度）。
- **D-02**：URL 小写英文 enum（`/anime/status/watching|done|plan`），页内中文标题。
- **D-03**：筛选入口放列表页顶部 PillRow。
- **D-09**：列表主页 `/anime`；slug 派生经 `slugOf(titleCn)`。
- **D-08（卡片部分）**：卡片上 star 取精简数字；详情页 Hero 才渲染 5 独立 star（见 03-03）。

## Verification

- `npx tsc --noEmit astro` 通过。
- `npx astro build` 构建成功；`/anime` 输出 HTML 含 AnimeCard 网格、PillRow、summary line。
- grep AnimeCard 输出含 `aspect-[2/3]`、`loading="lazy"`、`aria-label`。

- 浏览器（dev）访问 `/anime`：亮/暗主题均渲染；hover 卡上浮 2px；PillRow 当前项 `aria-current="page"`；空数据时显示 EmptyState 锁定文案。
