# 02-02 · 组件卡与两层链接（EntryCard 改造 + 5 个档案组件 + prose）

## 目标

在 02-01 数据层就绪基础上，把 `EntryCard.astro` 从「整张单 `<a>`」改造为「`<article.relative>` + 整卡内层 `<a href=详情>` + 绝对定位外 `<a href=外部 target=_blank rel=noopener>`」两层链接（零 JS / 零 Preact，解决 TOOL-05 安全外链与详情跳转互斥）。另新建 5 个轻组件（Breadcrumb / EmptyState / PillRow / TagCloud / ToolDetailNav），为 02-03 所有页面提供单一聚焦的视觉复用块。并在 `global.css` 追加 hand-rolled `.prose`（详情页 MDX 正文排版，不新装 `@tailwindcss/typography`）。

## 需求牵引

| Requirement | 本 plan 贡献 |
|---|---|
| TOOL-01 | EntryCard 整卡可点→详情、独立外链按钮、卡片内展示 title / summary（描述） |
| TOOL-05 | EntryCard 外链按钮 + 详情页外链按钮统一 `target="_blank" rel="noopener noreferrer"` + `aria-label="{工具名} 外链"`；布局覆盖避免 `<a>` 嵌套 |
